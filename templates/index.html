how it works

1. upload. the system accepts csv files with book data (10 to 500000+ rows).

2. training. random forest classifier analyzes dependencies between genre, rating, author popularity and other parameters.

3. results. you get model metrics and 4 charts with fund analytics.

4. prediction. enter parameters of a new book and the system will recommend the optimal number of copies to purchase.
